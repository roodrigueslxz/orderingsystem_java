# Incendius Cafe - Sistema de Pedidos

## Sobre o projeto
O **Incendius Cafe** é um sistema simples de pedidos desenvolvido em Java, executado via terminal, que simula a experiência de um cliente em um café temático inspirado em magia.

O usuário pode escolher uma bebida e uma sobremesa a partir de um menu interativo, e o sistema calcula automaticamente o valor total do pedido.

## Tema
O projeto possui uma proposta criativa baseada no universo de Harry Potter, trazendo nomes personalizados e divertidos para os produtos, como:

* Expresso Patronum
* Lattus Lumus
* Mocharm
* Capuccinox
* Frappus Maxima

E sobremesas como:

* Cheesecake de Frutas da Lua
* Brigadeiraccio
* Bolo de Mandrágora
* Muffin de Mirtilo Místico
  
## Funcionalidades
* Menu interativo no terminal
* Seleção de bebidas e sobremesas
* Cálculo automático dos preços
* Exibição do valor total do pedido
* Simulação de pagamento com entrada de senha

## Tecnologias utilizadas
* Java
* Entrada de dados com `Scanner`
* Estruturas condicionais (`switch-case`)

## Como executar
1. Certifique-se de ter o Java instalado
2. Compile o arquivo:
```
javac ordering.java
```
3. Execute o programa:
```
java ordering
```

Este projeto foi desenvolvido com fins educacionais, com foco no aprendizado de:
* Lógica de programação
* Entrada e saída de dados
* Estruturas condicionais
